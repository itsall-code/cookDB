编辑配置

1. 测试数据库连接

先连通后端 API，再验证所选数据库配置是否可用。

2. 备份数据库

源库 → 目标库，会覆盖目标库。执行前会弹出确认。

3. 单账号本地化

根据源账号 Field 生成或指定目标 Field，并写入目标 Redis。

4. 全库 acc 表本地化

自动读取 source Redis 中该 Hash 的全部字段，并写入 target Redis。

5. 删除指定 Keys / Tables

仅操作目标 Redis；确认弹窗会展示后端确认码。

6. 读取 Hash 字段并可视化

按需展开，查看 Hash 字段列表与解码结果。

7. 清空目标 DB

危险操作：清空目标 DB，需要输入完整后端确认码。
